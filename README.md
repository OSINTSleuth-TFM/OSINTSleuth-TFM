# OSINTSlush-TFM

OSINTSlush es una máquina virtual OSINT con un sistema operativo Ubuntu 20.04 💻

1. Frameworks 🎰
- OSRFramework
- Recon-ng
- Maltego

- OSINTool

2. Información DNS, correos electrónicos, dominios, etc 💻
- DNSEnum
- Infoga
- Spiderfoot
- ProtOSINT
- Fierce
- Holehe

3. Redes Sociales 📱
- Sherlock
- GitRecon

4. Descarga y Análisis De Archivos 💽
- MediaInfo
- ExifTool

5. Motores de búsqueda 🔎
- Pagodo
